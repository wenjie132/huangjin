#JAVA黄金矿工小游戏
##项目结构：
Bg类，背景类；

Gamewin类，游戏主类；

Gold类，金块类；

Line类，线类；

Object类，物体类；

Rock类，石头类；

##操作方式：
准备开始：开始界面点击鼠标右键；

开始游戏：当抓钩与金块或石块处于一条线中，点击鼠标左键，等待抓钩抓取物体；

使用药水：金块加速收取，石块直接摧毁；

通关成功：鼠标左键购买药水，鼠标右键跳过购买；

通关失败：点击鼠标左键重新开始，但刷新关卡数；
